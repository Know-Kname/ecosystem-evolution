# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do NOT** open a public issue
2. Email: knowledgeknight99@protonmail.com
3. Include a description and steps to reproduce
4. Allow 48 hours for initial response

## Scope

This repository manages device ecosystems and system configurations. Security concerns include:
- Unsafe PowerShell execution policies
- Docker configuration vulnerabilities
- WSL2 network exposure
- Credential management issues
